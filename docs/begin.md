# Installation and Usage Guidelines

## Installation
1. Clone the repository.
2. Run `tools/linux/setup.sh` (Linux) or `tools/windows/setup.bat` (Windows).
3. Follow the on-screen prompts to set up your profile.

## Usage
Use the `zennify` command with the following flags:
- `--activity`: Open Activity Tracker dashboard.
- `--flashcards`: Open Flashcards dashboard.
- `--todos`: Open Todos dashboard.
- `--pomodoro`: Open Pomodoro timer.
- `--shop`: Open Shop.
