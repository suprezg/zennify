# Zennify

## Abstract
Zennify is a gamified productivity ecosystem designed to help users track activity, master knowledge through flashcards, and manage time effectively.

## Objective
To provide a unified, privacy-focused, and low-overhead tool for modern productivity.

## Features
### Functional
- Interval Activity Tracking & Tagging
- Spaced Repetition Flashcards
- Time-Boxed Todos
- Pomodoro Timer
- Unified Economy & Shop

### Non Functional
- Privacy Focused
- Fast & Zero-Overhead
- Modern Aesthetics
- Portable & Transparent

## Specifications
### Requirements
- Python 3.12
### Dependencies
- Flet
